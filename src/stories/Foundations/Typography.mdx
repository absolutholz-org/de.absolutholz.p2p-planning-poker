# Typography

Our typography system is built on the **Inter** font family, prioritizing legibility and a modern aesthetic.

**Font Family:** `Inter, system-ui, Avenir, Helvetica, Arial, sans-serif`

## 📏 Font Sizes

Available as `--sys-font-size-*` CSS variables.

## 📝 Usage

- **XS (12px):** Labels, captions, micro-copy.
- **SM (14px):** Secondary text, small buttons.
- **MD (16px):** Body text, default inputs.
- **LG (18px):** Navigation elements, tooltips.
- **XL (20px):** Sub-headings, active labels.
- **2XL (24px):** Page titles, primary headings.
- **3XL (36px):** Large display titles.
