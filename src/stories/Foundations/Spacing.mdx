## 🌫️ Shadows

Shadow tokens for elevation. Root variable: `--sys-shadow-*`.

- `sm`: Subtle elevation for hover states.
- `md`: Default elevation for floating elements (Tooltips).
- `lg`: High elevation for primary overlays (Popovers).
- `xl`: Maximum elevation for modal dialogs.
